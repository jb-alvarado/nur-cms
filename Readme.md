# nur-cms

A simple and fast content management system.

## Features

- Easy content editing
- User authentication
- RESTful API

## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/jb-alvardo/nur-cms.git
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start backend server:
    ```bash
    cargo run
    ```
4. Start frontend:
    ```bash
    npm run dev
    ```
